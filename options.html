Reasy Options

Words per minute
Higher the value, faster the words appear on the screen. Start with a lower value and them proceed up
Punctuation pause in ms
Pause reading at puctuation characters. Set this to 0 if you don't want to pause at punctuations
Font size in px
How big you want the reading font size to be
Background Color
CSS hex code or name for background color
Foreground Color
CSS hex code or name for foreground color
Z Index
Increase if any page elements obstruct reading

Keyboard shortcuts are case-sensitive. Also make sure you choose keyboard shortcuts that are least conflicting with the website page. For example Google Reader uses 'r' key to refresh page. You might not want to pick the same key for launching Reasy.

Reasy can skip some words backwards and forwards as you read and you can choose keys to do that. It can skip sentences back and forwards too. There's also an option to read slow until you get back to the original position, if you skip some words backwards.
